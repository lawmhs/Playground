Todo List

How to use :

Use the input on add to add new items to your todo list.

The Toggle All button will either mark all of your todo items to be done, or, if they're all done, then it will mark all of them not done!

To change your todo items, just click the text of the item and an input bar will be shown You can use that to change the todo item.
